# KidsGuide
Reactive web application using Vue.js, Node.js and PostgreSQL.

Website: https://kynangsong.herokuapp.com

+ How to run on localhost:
	+ cd `frontend`
	+ npm run `build`
	+ copy `dist` folder to `backend`
	+ cd `backend`
	+ npm run `start`
